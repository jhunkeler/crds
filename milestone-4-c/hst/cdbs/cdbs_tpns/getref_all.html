Explanation of Results

Rootname
The name of the observation set using this reference file, as enterd on the form.
Keyword
The name of the keyword in the science data file header.
Value
The new value of the keyword.
Filename
The file in which the keyword occurs.
Operation
The operation to be performed on the file: insert a new keyword, update an existing keyword or delete a keyword.
Change
Has this keyword value changed from its original value? Yes or no.
Source
The database table that is the source of this information
