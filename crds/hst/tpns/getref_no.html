Reference File Search

Enter the name of an observation set to retrieve the names of the best reference files and switches to use with an observation. If you wish to retrieve the reference files for more than one observation, enter the observation set names separated by blanks, commas, or newlines. Wild card characters are not allowed in the list.

This screen can produce two types of output: text and html. The text output can be saved and used with the upref task to update the header keywords in an observation.
